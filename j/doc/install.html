Installation

To compile from source, refer to the building documentation.

Java 1.6 or later is required, but Oracle Java 1.6 is recommended.

Support for platforms other than Linux and Windows is purely accidental.

Windows

Download the j-0.23.0-win.zip archive, unzip, and double-click the J.exe.

Mac OS X

Download the j-0.23.0-mac.zip archive, unzip, and double-click the J.app.

Other platforms

Simply untar or unzip the binary distribution, cd into the j directory, and execute the jar file using java: 

    $ tar xvfz j-0.23.0-bin.tar.gz
    $ cd j-0.23.0
    $ java -jar j.jar
Or on Windows: 
    c:\> unzip j-0.23.0-bin.zip
    c:\> cd j-0.23.0
    c:\j-0.23.0> java -jar j.jar
You should be able to bring up j's help from within the editor by pressing F1. If this doesn't work right out of the box, you may need to create a preferences file and add a line to it to tell j where the documentation is located: 
    docPath=C:\\j\\doc
docPath should point to the directory containing j's HTML documentation (including the file you're reading now). On Windows, you'll need to use double backslashes in your filenames, as shown.

To use the jdb debugger, you'll need to include tools.jar from the JDK on the classpath. Read the debugger documentation for more information.
